//This document list all kind of useful information regarding the pet feeder

Components:
-esp12e + esp12e motorshield
-12v -> 5v convertor
-16*2 LCD with I2C
-DS3231 RTC
-Rotary encoder
- 12v 20rps DC motor (10kg.cm)

Platform: 
Platformio on Visual code

Pin mapping:
    shield
        D1-> PWMA motor A 
        D2-> PWMA motor B
        D3 -> direction motor A 
        D4 -> direction motor B
    LCD
